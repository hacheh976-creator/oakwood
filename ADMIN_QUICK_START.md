# OAKWOOD Admin Panel - Quick Start Guide

## 🔐 Logging Into Admin Panel

1. **From Role Selection Screen:**
   - Tap the Oakwood logo **5 times quickly**
   - You'll be redirected to admin login

2. **Enter Password:**
   - Password: `admin123`
   - Click "Login"

3. **You're In!**
   - Welcome to the admin dashboard

---

## 📦 Managing Products (Storefront)

### Add a New Product:

1. **Navigate:**
   - Admin Dashboard → Click "Storefront"
   - Click "Products"
   - Click "+ Add Product" button

2. **Fill in Details:**
   - **Name:** e.g., "Modern Oak Dining Table"
   - **Category:** Select from dropdown (Tables, Chairs, etc.)
   - **Price:** e.g., 25000 (in your currency)
   - **Stock:** e.g., 5
   - **Image URL:** Paste a valid image URL
   - **Gallery URLs:** Optional - add more images
   - **Description:** Write product details
   - **Featured:** Check if you want it on homepage

3. **Save:**
   - Click "Add Product"
   - Product is now live on customer storefront!

### Edit Existing Product:

1. Click the **Edit** icon (pencil) next to any product
2. Update any fields
3. Click "Update Product"
4. Changes appear immediately on storefront

### Delete Product:

1. Click the **Trash** icon next to any product
2. Confirm deletion
3. Product is removed from database

---

## 🏷️ Managing Categories

### Add a Category:

1. **Navigate:**
   - Admin Dashboard → Storefront → Categories
   
2. **Add New:**
   - Click "+ Add Category"
   - Enter category name (e.g., "Dining Tables")
   - Add image URL
   - Save

3. **Result:**
   - Category appears in customer navigation
   - Products can be assigned to this category

## 🎨 Managing Home Banners

## 💰 Recording Sales

### Add a Sale:

1. **Navigate:**
   - Admin Dashboard → Sales
   - Click "+ Add Sale"

2. **Fill in Details:**
   - Product Name
   - Customer Name
   - Customer Phone
   - Quantity
   - Amount (total price)
   - Payment Method (Cash, Card, Bank Transfer)
   - Payment Type:
     - **Full:** Customer paid everything
     - **Credit:** Partial payment
     - **Discount:** Applied discount
   - Discount % (if applicable)

3. **Save:**
   - Sale is recorded in database
   - Appears in sales reports

---

## 📋 Managing Orders

### Add a Custom Order:

1. **Navigate:**
   - Admin Dashboard → Orders
   - Click "+ Add Order"

2. **Order Details:**
   - Product Name
   - Customer Info (name, phone, address)
   - Quantity
   - Total Amount
   - Advance Payment (deposit)
   - Expected Delivery Date
   - Order Type:
     - **Custom:** Made to order
     - **Stock:** From existing inventory
   - Status: Pending, In Production, Ready, or Delivered

3. **Track Order:**
   - Update status as order progresses
   - View all orders in one place

## 💵 Tracking Expenses

### Record an Expense:

## 🖼️ Working with Images

### Option 1: Use Image URLs

1. **Upload image to:**
   - [Imgur.com](https://imgur.com) (Free, no account needed)
   - [ImgBB.com](https://imgbb.com) (Free)
   - Your own website

2. **Copy direct image URL:**
   - Right-click image → "Copy Image Address"
   - Paste into product image field

### Option 2: Upload Directly (Advanced)

1. In product form, look for file upload button
2. Select image from your computer
3. Wait for upload to complete
4. URL will be auto-filled

## 🆘 Common Issues

### Can't Login:
- Make sure password is exactly: `admin123`
- Tap logo 5 times (not more, not less)

### Image Not Showing:
- Check if URL is valid (paste in browser)
- Make sure it's a direct image link (ends in .jpg, .png)
- Try using Imgur for reliable hosting

### Changes Not Saving:
- Check internet connection
- Look for error messages
- Check browser console (F12)

### Product Not Appearing on Storefront:
- Make sure stock > 0
- Check if category exists
- Verify all required fields are filled
